# ⚖️ LegalEase - Premium Lawyer Consultation Platform (Client)

LegalEase is a high-end, modern, and minimalist digital experience designed for connecting clients with top-rated legal specialists. The platform features an Apple-style, dark-themed aesthetic with fluid animations and seamless micro-interactions.

## ✨ Features
- **Premium UI/UX**: Minimalist dark theme with a sleek slate and emerald/cyan color palette.
- **Top Rated Specialists**: Dynamic showcase of featured lawyers with real-time availability status.
- **Advanced Search & Browse**: Filter and find legal professionals based on specialization and hourly rates.
- **Smooth Animations**: Integrated micro-interactions for a premium feel.
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop viewports.

## 🛠️ Tech Stack
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **State Management & Fetching**: React Hooks (useState, useEffect)

## 🚀 Getting Started

### 1. Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed.

### 2. Installation
Clone the repository and navigate to the client folder, then run:
```bash
npm install
3. Environment Variables
Create a .env.local file in the root of your client directory and add your backend API URL:

Code snippet
NEXT_PUBLIC_API_URL=[https://legalease-server-neon.vercel.app](https://legalease-server-neon.vercel.app)
4. Running Locally
Start the development server:

Bash
npm run dev
Open http://localhost:3000 with your browser to see the result.
